## Unity 项目设置

### 1. 创建 Unity 项目
- 打开 Unity Hub
- 创建新的 2D 项目，命名为 "林间灵物信使"
- 选择此文件夹作为项目位置

### 2. 导入脚本文件
- 将 Assets/Scripts 文件夹中的所有脚本拖入 Unity 项目的相应位置

## 场景搭建步骤

### 森林主场景 (ForestScene)

#### 1. 创建游戏管理器
```
GameObject: GameManager
  - 添加组件: GameManager
  - 添加组件: InputManager
```

#### 2. 创建主相机
```
Main Camera:
  - Tag: MainCamera
  - 添加组件: CameraFollow
  - CameraFollow 设置:
    * Smooth Time: 0.25
    * Offset: (0, 2, -10)
```

#### 3. 创建森林环境
```
GameObject: ForestEnvironment
  - 添加组件: ForestGenerator
  - ForestGenerator 设置:
    * Ground Width: 50
    * Tree Count: 20
    * Bush Count: 30
    * Grass Count: 50
```

#### 4. 创建地面
```
GameObject: Ground
  - Tag: Ground
  - Layer: Ground (需要创建新Layer)
  - 添加组件: SpriteRenderer
    * Color: 棕色 (0.4, 0.26, 0.13)
  - 添加组件: BoxCollider2D
    * Size: (50, 1)
  - Position: (0, 0, 0)
```

#### 5. 创建玩家预制体
```
GameObject: Player
  - Tag: Player
  - Layer: Player
  - Position: (0, 2, 0)
  - Scale: (1, 1, 1)

组件配置:
  1. Rigidbody2D:
     - Gravity Scale: 3
     - Freeze Rotation: Z
     - Collision Detection: Continuous

  2. CapsuleCollider2D:
     - Size: (0.8, 1.6)
     - Offset: (0, 0)

  3. PlayerController:
     - Move Speed: 8
     - Jump Force: 16
     - Ground Layer: 选择 Ground Layer
     - Ground Check Radius: 0.2

子对象:
  GameObject: Graphics (作为Player的子对象)
    - 添加组件: SpriteRenderer
      * 导入小动物精灵图片后设置
    - Position: (0, 0, 0)
```

## 控制说明

### 基础操作
- **A / ←**: 向左移动
- **D / →**: 向右移动
- **空格 / W / ↑**: 跳跃
- **ESC**: 暂停/继续游戏

### 玩家控制器特性
- ✅ 平滑加速/减速
- ✅ 土狼时间 (Coyote Time)
- ✅ 跳跃缓冲
- ✅ 可变跳跃高度
- ✅ 自动地面检测点创建
- ✅ 角色朝向翻转

## Layer 设置（重要！）

请在 Unity 中创建以下 Layers:
1. **Ground** - 用于地面检测
2. **Player** - 玩家层级
3. **Environment** - 环境物体层级

## 导入资源建议

### 美术资源
- **角色**: 卡通风格小动物（狐狸、松鼠、兔子等）
- **背景**: 森林分层背景（远景山脉、中景树木、近景灌木）
- **植被**: 树木、灌木、草丛的独立精灵
- **平台**: 各种样式的地面平台

### 动画状态机建议
创建 Animator Controller，包含以下状态:
- Idle (待机)
- Walk (行走)
- Jump (跳跃上升)
- Fall (下落)
- Land (落地)

Animator 参数:
- Speed (float)
- IsGrounded (bool)
- VerticalVelocity (float)

## 常见问题

### 玩家无法跳跃
- 检查 Ground Layer 是否正确设置
- 检查 GroundCheck 位置是否在角色脚下
- 检查 Rigidbody2D 的 Gravity Scale

### 移动不流畅
- 调整 acceleration 和 deceleration 参数
- 检查 Rigidbody2D 的 Interpolation 设置

### 相机抖动
- 增大 Smooth Time 参数
- 确保 CameraFollow 在 LateUpdate 中执行
